# Xca Agent

> 一个跨平台、可灵活对接本地/云端大模型的智能编程助手

## 项目概述

Xca Agent 是一个受 Claude Code 启发的命令行 AI 助手，支持灵活切换本地部署模型（Ollama）和云端 API（OpenAI、Claude、OpenRouter 等），同时支持 Windows 和 macOS 双平台。

## 核心特性预览

- 🔌 **灵活的 LLM 接入** - 一键切换本地/云端模型
- 🖥️ **跨平台支持** - Windows & macOS 原生体验
- 🛠️ **强大的工具系统** - 文件操作、命令执行、代码搜索
- 💬 **上下文感知** - 智能管理长对话和大文件
- 🔒 **隐私优先** - 本地模型支持，数据不出本机

## 设计哲学

> "好的设计是尽可能少的设计。" — Dieter Rams

Xca Agent 追求：
1. **简单性** - 学习成本低，功能一目了然
2. **灵活性** - 不强推特定模型，让用户自己选择
3. **透明性** - 每一步操作都可见、可审查、可撤销
4. **渐进式** - 从简单场景开始，逐步解锁高级能力
